**🔒 DNI ** **Env** **ío seguro**  
Aplicación web para proteger las imágenes de tu DNI español antes de enviarlas por correo electrónico, WhatsApp o cualquier trámite online.  
**Todo el procesamiento ocurre en tu dispositivo. Ningún dato sale de tu teléfono o navegador.**

**¿Para qué sirve?**  
Cuando tienes que enviar una copia de tu DNI (para un banco, alquiler, trámite administrativo, etc.) es recomendable protegerlo para evitar usos fraudulentos. Esta app te permite:  
- Añadir una **marca de agua** con el destinatario y la fecha  
- **Pixelar** los campos más sensibles (firma, número de soporte, MRZ, etc.)  
- Descargar el resultado como **JPG** o  **PDF** listo para enviar

**Campos recomendados para ocultar**  
**🔴 Ocultar siempre**  
- Firma  
- Número de soporte  
- Fechas de validez y emisión  
- Número CAN / Código CLI  
- Código IDESP / equipo de expedición  
- Líneas MRZ/OCR (las tres líneas del reverso)  
**🟡 Ocultar si no es necesario**  
- Sexo  
- Nacionalidad  
- Fecha de nacimiento  
- Domicilio  
- Lugar de nacimiento / nombre de padres  
**🟢 Puede quedar visible**  
- Fotografía  
- Nombre y apellidos  
- Número de DNI  
![](data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAAAnEAAAACCAYAAAA3pIp+AAAABmJLR0QA/wD/AP+gvaeTAAAACXBIWXMAAA7EAAAOxAGVKw4bAAAANElEQVR4nO3OQQmAABRAsSdYxKa/jL0MIR7FCt5E2BJsmZmt2gMA4C+Otbqr8+sJAACvXQ85SAYUQNBTfQAAAABJRU5ErkJggg==)  
**Cómo usar**  
1. **Sube** las fotos del anverso y reverso de tu DNI  
2. Pulsa **✏️ Pixelar datos** en cada cara para ocultar campos sensibles  
- Arrastra para seleccionar zonas  
- Usa pinch-to-zoom o los botones para acercar y ver mejor el texto  
3. Escribe en **Marca de agua** para quién es la copia y la fecha (ej: SOLO USO OFICIAL · 28/04/2026)  
4. Ajusta el **color**,  **opacidad** y  **tamaño** de la marca  
5. Descarga el resultado como **JPG** o  **PDF**  
![](data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAAAnEAAAACCAYAAAA3pIp+AAAABmJLR0QA/wD/AP+gvaeTAAAACXBIWXMAAA7EAAAOxAGVKw4bAAAANUlEQVR4nO3OMQ2AABAAsSNhwgJOUPcjIpnRgQU2QtIq6DIze3UGAMBf3Gu1VcfXEwAAXrseaJEEL8XMiYMAAAAASUVORK5CYII=)  
**Instalar como app en Android**  
1. Abre el enlace en **Chrome para Android**  
2. Pulsa el menú (⋮) → **"Añadir a pantalla de inicio"**  
3. La app aparece como icono en tu escritorio  
4. Funciona sin conexión a internet  
![](data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAAAnEAAAACCAYAAAA3pIp+AAAABmJLR0QA/wD/AP+gvaeTAAAACXBIWXMAAA7EAAAOxAGVKw4bAAAANklEQVR4nO3OMQ2AABAAsSNBACP6MMH6NpGACyywEZJWQZeZ2aszAAD+4l6rrTq+ngAA8Nr1AL+6BElk4wV6AAAAAElFTkSuQmCC)  
**Tecnología**  
- HTML + CSS + JavaScript puro — sin frameworks, sin dependencias externas  
- [jsPDF para exportación a PDF](https://github.com/parallax/jsPDF "https://github.com/parallax/jsPDF")  
- Canvas API para procesado de imágenes  
- PWA con Service Worker para uso offline  
- **Sin servidor** — todo ocurre en el navegador del usuario
